# Desenvolvimento com IA assistida

1. Oriente-se com m1nd e Graphify antes de busca ampla; confirme decisões no código, testes e runtime.
2. Separe implementação e revisão independente. Revisão por agente é evidência técnica, não aprovação humana.
3. Não envie dados clínicos, credenciais reais ou dados pessoais a prompts, ferramentas ou logs.
4. Restrinja mudanças ao critério de aceite; registre dívida P2 em vez de aumentar escopo.
5. Após mudança relevante execute `graphify update .` e mantenha documentação, 5W2H, riscos e rollback coerentes.
